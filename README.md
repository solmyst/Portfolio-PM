# Anush Gupta's Product Manager Portfolio

A premium, highly interactive portfolio designed as a Product Requirements Document (PRD), built to showcase real product thinking, execution, and impact. 

- **What it is:** A Google-Docs-styled interactive PRD showcasing PM capabilities, real internship case studies, and side project concepts.
- **Why the PRD format:** I believe PM portfolios should demonstrate actual product skills. Since writing clear, structured PRDs is a core PM capability, this portfolio is formatted as a live, interactive product spec itself.
- **Tech Stack:** React.js, Tailwind CSS, Framer Motion, and deployed on Vercel.
- **Live Site:** [https://dev.anushgupta.tech](https://dev.anushgupta.tech)

*No generic feature lists, no buzzword salads—just transparent execution, real metrics, and product concepts I am actively exploring.*
